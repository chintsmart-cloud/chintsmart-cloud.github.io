（通知）¶

设备连接状态变更推送（SERVICE）¶

通知TOPIC：

  • /sys/${product_id}/${device_id}/app/down/thing/status

通知方式：

  • MQTT

通知示例

{
    "error_code": 200,
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "device_id": "53102fb4bf1044ed8b0ba36c",
    "data": {
        "status": 1
    }
}

通知数据说明

参数名

类型

说明

status

integer

设备状态, 0:未激活; 1:在线; 2:离线; 3:禁用; 4:已激活

注解

message_id 和 error_code 参见 公共参数说明-参数列表

设备事件推送¶

通知TOPIC：

  • /sys/${product_id}/${device_id}/app/down/thing/events

通知方式：

  • MQTT

通知示例

{
    "error_code": 200,
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "device_id": "53102fb4bf1044ed8b0ba36c",
    "data": {
        "user_id": "5124b5d86041c7dca81916",
        "event_id": "5124b5d86041c7dca81917",
        "event_type": 0,
        "properties": {},
        "code": 0,
        "gmt_create": 1499681215000
    }
}

通知数据说明

参数名

必选

类型

user_id

string

用户的唯一id

event_id

string

当条事件的唯一id

event_type

integer

事件类型, 0:属性上报(info); 1:系统上报(waring) 2:故障上报(error);

properties

object

物模型, 和品类id配对, 参考 公共参数说明-物模型

code

integer

参见 公共参数说明-错误码字典

gmt_create

long

创建时间戳(毫秒)

注解

message_id 和 error_code 参见 公共参数说明-参数列表

固件升级进度推送¶

通知TOPIC：

  • /sys/${product_id}/${device_id}/app/down/ota/device/forward

通知方式：

  • MQTT

通知示例

{
    "error_code": 200,
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "device_id": "53102fb4bf1044ed8b0ba36c",
    "data": {
        "sdk_version": "1.0.0",
        "src_version": "1.0.0",
        "dest_version": "2.0.0",
        "step": 50,
        "upgrade_status": 1
    }
}

通知数据说明

参数名

类型

说明

sdk_version

string

SDK版本

src_version

string

表示最近一次升级的源版本

dest_version

string

表示最近一次升级目标版本

step

integer

表示固件升级进度信息，可取值包括， -1：表示升级失败，-2：表示下载失败，-3：表示校验失败，-4：表示烧写失败， [1，100] 之间的数字：表示升级进度百分比

upgrade_status

integer

表示升级结果，可取值包括，0：待升级或待确认，1：升级中，2：升级异常，3：升级失败，4：升级完成

注解

message_id 和 error_code 参见 公共参数说明-参数列表
