更新设备信息接口¶

请求URL：

  • https://api.chintsmart.com/device/update

请求方式：

  • POST

请求示例

{
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "params": {
        "access_token": "xxxxxx",
        "device_id": "53102fb4bf1044ed8b0ba36c",
        "update_model": 0,
        "device_secret": "53102fb4bf1044ed8b0ba36a",
        "device_name": "xxxxxx",
        "sn": "xxxxxx",
        "firmware_version": "xxxxx",
        "mac_addr": "xxxxx",
        "net_addr": "xxxxx",
        "nick_name": null,
        "properties": {}
    }
}

请求参数说明

参数名

必选

类型

说明

access_token

是

string

访问token

device_id

是

string

设备的唯一id

update_model

是

integer

更新设备方式, 0:全部信息; 1:设备密钥; 2:设备名称; 3:设备序列号; 4:固件版本; 5:物理地址; 6:网络地址; 7:设备昵称; 8:具体属性

device_secret

否

string

设备密钥, 当 update_model 为 0 或 1 时 必填

device_name

否

string

设备名称, 当 update_model 为 0 或 2 时 必填

sn

否

string

设备序列号, 当 update_model 为 0 或 3 时 必填

f irmware_version

否

string

设备固件版本, 当 update_model 为 0 或 4 时 必填

mac_addr

否

string

设备物理地址, 当 update_model 为 0 或 5 时 必填

net_addr

否

string

设备网络地址, 当 update_model 为 0 或 6 时 必填

nick_name

否

string

设备昵称, 当 update_model 为 0 或 7 时 必填

properties

否

object

具体属性对象, 当 update_model 为 0 或 8 时 必填

返回示例

{
    "error_code": 200,
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "data": null
}

返回数据说明

  • 空值

备注

  • message_id 和 error_code 参见 公共参数说明

更新设备记录接口¶

请求URL：

  • https://api.chintsmart.com/device/update_record

请求方式：

  • POST

请求示例

{
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "params": {
        "access_token": "xxxxxx",
        "record_id": "53102fb4bf1044ed8b0ba36c",
        "update_model": 0,
        "level": 0,
        "status": 1
    }
}

请求参数说明

参数名

必选

类型

说明

access_token

是

string

访问token

record_id

是

string

记录的唯一id

update_model

是

integer

更新设备记录方式, 0:全部; 1:记录等级; 2:记录状态

level

否

integer

当 update_model 为 0 或 1 时 必填 , 记录的严重等级, 0:普通(info); 1:警告(waring); 2:严重(fatal);

status

否

integer

当 update_model 为 0 或 2 时 必填 , 记录的状态, 0:全部; 1:可用; 2:归档; 3:未读; 4:已读

返回示例

{
    "error_code": 200,
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "data": null
}

返回数据说明

  • 空值

备注

  • message_id 和 error_code 参见 公共参数说明
