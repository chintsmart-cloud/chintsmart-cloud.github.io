设备服务¶

绑定设备接口¶

请求URL：

  • https://api.chintsmart.com/device/bind

请求方式：

  • POST

请求示例

{
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "user_id": "53102fb4bf1044ed8b0ba36c",
    "params": {
        "access_token": "xxxxxx",
        "device_id": "53102fb4bf1044ed8b0ba36c",
        "device_secret": "53102fb4bf1044ed8b0ba36a",
        "mac_addr": "xxxxx",
        "sdk_version": "1.0.0",
        "version": "1.0.0",
        "nick_name": "xxxx",
        "net_addr": "xxxxx",
        "status": 1,
        "owned": 1,
        "group_id": "xxxxxxx",
        "is_sub_device": false,
        "product_id": "53102fb4bf1044ed8b0ba36c",
        "category_id": 1
    }
}

请求参数说明

参数名

必选

类型

说明

access_token

是

string

访问token

device_id

是

string

设备的唯一id

device_secret

否

string

设备密钥

mac_addr

是

string

设备物理地址

sdk_version

是

string

SDK版本

version

是

string

设备固件版本

nick_name

否

string

设备昵称

net_addr

否

string

设备网络地址

status

是

integer

设备状态, 0:未激活; 1:在线; 2:离线; 3:禁用; 4:已激活

group_id

否

integer

空间编号，如家ID、房间ID等

is_sub_device

否

boolean

是否为子设备。归属在某一个网关设备下的设备表示为子设备。 取值为：true （表示获取子设备列表）；false （表示获取直连设备列表）

owned

否

integer

用户获取设备的方式, 0:被分享的设备; 1:拥有的设备

product_id

是

string

产品的唯一id

category_id

是

string

品类id, 和物模型配对, 参考 公共参数说明-物模型

返回示例

{
    "error_code": 200,
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "data": null
}

返回数据说明

  • 空值

注解

message_id 和 error_code 参见 公共参数说明-参数列表

更新设备基本信息接口¶

请求URL：

  • https://api.chintsmart.com/device/update_info

请求方式：

  • POST

请求示例

{
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "user_id": "53102fb4bf1044ed8b0ba36c",
    "params": {
        "access_token": "xxxxxx",
        "device_id": "53102fb4bf1044ed8b0ba36c",
        "nick_name": "xxxx",
        "owned": 1,
        "group_id": "xxxxxxx",
        "is_sub_device": false
    }
}

请求参数说明

参数名

必选

类型

说明

access_token

是

string

访问token

device_id

是

string

设备的唯一id

nick_name

否

string

设备昵称, 当 update_model 为 0 或 1 时 必填

owned

否

integer

用户获取设备的方式, 0:被分享的设备; 1:拥有的设备, 当 update_model 为 0 或 2 时 必填

group_id

否

integer

要查询的空间，家ID、房间ID等, 当 update_model 为 0 或 3 时 必填

is_sub_device

否

boolean

是否为子设备。归属在某一个网关设备下的设备表示为子设备。 取值为：true （表示获取子设备列表）；false （表示获取直连设备列表） , 当 update_model 为 0 或 4 时 必填

返回示例

{
    "error_code": 200,
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "data": null
}

返回数据说明

  • 空值

注解

message_id 和 error_code 参见 公共参数说明-参数列表

查询设备事件接口¶

请求URL：

  • https://api.chintsmart.com/device/query_record

请求方式：

  • POST

请求示例

{
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "user_id": "53102fb4bf1044ed8b0ba36c",
    "params": {
        "access_token": "xxxxxx",
        "query_model": 0,
        "page_no": 1,
        "page_size": 100,
        "start": 1499681215000,
        "end": 1499681216000,
        "device_id": "53102fb4bf1044ed8b0ba36c",
        "user_id": "53102fb4bf1044ed8b0ba36c",
        "event_type": 0
    }
}

请求参数说明

参数名

必选

类型

说明

access_token

是

string

访问token

query_model

是

integer

查询方式, 0:设备id; 1:用户id;

page_no

是

integer

当前页号, 从1开始

page_size

是

integer

分页大小，大于等于1，小于等于100

start

否

long

查询历史开始时间戳, 距离当前时间较远的时间戳

end

否

long

查询历史结束时间戳, 距离当前时间较近的时间戳

device_id

否

string

设备的唯一id, 当 query_model , 为 0 时 必填

user_id

否

string

注册应用或用户的唯一id, 当 query_model 为 1 时 必填

event_type

否

integer

事件类型, 不填时为所有, 0:属性上报; 1:故障及系统上报;

返回示例

{
    "error_code": 200,
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "data":{
        "total": 2,
        "page_no": 1,
        "page_size": 100,
        "events_list": [
            {
                "device_id": "53102fb4bf1044ed8b0ba36c",
                "user_id": "5124b5d86041c7dca81916",
                "event_id": "5124b5d86041c7dca81917",
                "event_type": 0,
                "properties": {},
                "code": 0,
                "gmt_create": 1499681215000
            },
            {
                "device_id": "53102fb4bf1044ed8b0ba36c",
                "user_id": "5124b5d86041c7dca81916",
                "event_id": "5124b5d86041c7dca81917",
                "event_type": 0,
                "properties": {},
                "code": 0,
                "gmt_create": 1499681215000
            }
        ]
    }
}

返回数据说明

参数名

必选

类型

total

integer

云端总数

page_no

integer

当前页号, 从1开始

page_size

integer

分页大小，大于等于1，小于等于100

events_list

array

事件列表

.device_id

string

设备的唯一id

.user_id

string

用户的唯一id

.event_id

string

当条事件的唯一id

.event_type

integer

事件类型, 0:属性上报(info); 1:系统上报(waring) 2:故障上报(error);

.properties

object

物模型, 和品类id配对, 参考 公共参数说明-物模型

.code

integer

参见 公共参数说明-参数列表

.gmt_create

long

创建时间戳(毫秒)

注解

message_id 和 error_code 参见 公共参数说明-参数列表
