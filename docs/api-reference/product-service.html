产品管理¶

查询品类列表接口¶

请求URL：

  • https://api.chintsmart.com/product/query_category

请求方式：

  • POST

请求示例

{
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "params": {
        "access_token": "xxxxxx",
        "client_id": "53102fb4bf1044ed8b0ba36c"
    }
}

请求参数说明

参数名

必选

类型

说明

access_token

是

string

访问token

client_id

是

string

注册应用的唯一id

返回示例

{
    "error_code": 200,
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "data": {
        "client_id": "53102fb4bf1044ed8b0ba36c",
        "category_list": [
            {
                "category_id": 0,
                "category_name": "xxxx",
                "category_icon": "https:/www.a.com/1.jpg"
            },
            {
                "category_id": 1,
                "category_name": "xxxx",
                "category_icon": "https:/www.a.com/1.jpg"
            },
            {
                "category_id": 2,
                "category_name": "xxxx",
                "category_icon": "https:/www.a.com/1.jpg"
            }
        ]
    }
}

返回数据说明

参数名

类型

说明

client_id

string

注册应用的唯一id

category_list

array

品类列表

.category_id

integer

品类id

.category_name

string

品类名称

.category_icon

string

品类图标地址

备注

  • message_id 和 error_code 参见 公共参数说明

查询产品接口¶

请求URL：

  • https://api.chintsmart.com/product/query

请求方式：

  • POST

请求示例

{
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "params": {
        "access_token": "xxxxxx",
        "query_model": 0,
        "product_id": "53102fb4bf1044ed8b0ba36c",
        "product_key": "53102fb4bf1044ed8b0ba36b"
    }
}

请求参数说明

参数名

必选

类型

说明

access_token

是

string

访问token

query_model

是

integer

查询方式, 0:产品id; 1:产品key

product_id

否

string

产品的唯一id, 当 query_model 为 0 时 必填

product_key

否

string

产品key, 当 query_model 为 1 时 必填

返回示例

{
    "error_code": 200,
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "data": {
        "product_id": "53102fb4bf1044ed8b0ba36c",
        "product_key": "53102fb4bf1044ed8b0ba36b",
        "product_secret": "53102fb4bf1044ed8b0ba36a",
        "product_name": "xxxxx",
        "category_id": 0,
        "vendor_type": 0,
        "net_type": 0,
        "product_icon": "https:/www.a.com/1.jpg",
        "status": 1,
        "creator": "1bd5d003-31b9-476f-ad03-71d471922820",
        "modifier": "1bd5d003-31b9-476f-ad03-71d471922820",
        "gmt_create": 1499681215000,
        "gmt_modified": 1499681216000
    }
}
