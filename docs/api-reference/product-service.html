产品服务¶

查询品类列表接口¶

请求URL：

  • https://api.chintsmart.com/product/query_category

请求方式：

  • POST

请求示例

{
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "user_id": "53102fb4bf1044ed8b0ba36c",
    "params": {
        "access_token": "xxxxxx",
        "client_id": "53102fb4bf1044ed8b0ba36c"
    }
}

请求参数说明

参数名

必选

类型

说明

access_token

是

string

访问token

client_id

是

string

注册应用的唯一id

返回示例

{
    "code": 200,
    "message_id": "1bd5d003-31b9-476f-ad03-71d471922820",
    "data": {
        "client_id": "53102fb4bf1044ed8b0ba36c",
        "category_list": [
            {
                "category_id": 0,
                "category_name": "xxxx",
                "category_icon": "https:/www.a.com/1.jpg"
            },
            {
                "category_id": 1,
                "category_name": "xxxx",
                "category_icon": "https:/www.a.com/1.jpg"
            },
            {
                "category_id": 2,
                "category_name": "xxxx",
                "category_icon": "https:/www.a.com/1.jpg"
            }
        ]
    }
}

返回数据说明

参数名

类型

说明

client_id

string

注册应用的唯一id

category_list

array

品类列表

.category_id

integer

品类id

.category_name

string

品类名称

.category_icon

string

品类图标地址

注解

message_id 和 code 参见 公共参数说明-参数列表
