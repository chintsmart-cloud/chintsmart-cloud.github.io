授权登陆接口¶

请求URL：

  • https://api.chintsmart.com/token/login

请求方式：

  • GET

请求示例

  • ?redirect_uri=https%3a%2f%2fapi.chintsmart.com%2ftoken%2fcallback%3fskill_id%3d11111%26token%3dxxxxx

  • &client_id=XXXXXXXXX

  • &response_type=code

  • &state=111

请求参数说明

参数名

必选

类型

说明

redirect_uri

是

string

回调地址, 如果没有可填写none

client_id

是

string

注册应用的唯一id

response_type

是

string

返回类型, code

state

是

string

自定义状态值

返回示例

{
    "error_code": 200,
    "access_token": "xxxxx",
    "refresh_token": "xxxxxx",
    "expires_in": 172800,
    "openid": "53102fb4bf1044ed8b0ba36c"
}

返回数据说明

参数名

类型

说明

access_token

string

访问token

refresh_token

string

进行刷新access_token的刷新token

expires_in

long

访问token和刷新token的过期时间时长(秒)

openid

string

用户或设备的唯一标识

备注

  • error_code 参见 公共参数说明

访问&刷新token接口¶

请求URL：

  • https://api.chintsmart.com/token/access

请求方式：

  • POST

访问-请求示例

  • ?grant_type=authorization_code

  • &client_id=53102fb4bf1044ed8b0ba36c

  • &client_secret=xxxxxx

  • &code=xxxxxx

  • &redirect_uri=none

刷新-请求示例

  • ?grant_type=refresh_token

  • &client_id=53102fb4bf1044ed8b0ba36c

  • &client_secret=xxxxxx

  • &refresh_token=xxxxxxxx
