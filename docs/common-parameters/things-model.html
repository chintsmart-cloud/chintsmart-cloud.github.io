0 开关¶

请求示例

{
    "category_id": 0,
    "properties": {
        "0": {
            "power_state": true,
            "timing_timer": [
                {
                    "enable": true,
                    "repeat": "everyday",
                    "start": {
                        "timer": "8:30",
                        "power_state": true
                    },
                    "end": {
                        "timer": "23:30",
                        "power_state": false
                    }
                }
            ],
            "countdown_timer": [
                {
                    "after_seconds": 3600,
                    "power_state": true
                }
            ]
        },
        "1": {
            "power_state": true,
            "countdown_timer": [
                {
                    "remaining": 3600,
                    "power_state": true
                }
            ]
        }
    }
}

属性说明

参数名

类型

说明

power_state

boolean

开关的状态, 开: true , 关: false

timing_timer

object

定时器

.enable

boolean

定时器是否开启, 开: true : 开 , 关: false

.repeat

string

重复执行的方式, once : 执行一次; everyday : 每天执行;

1,2,3,4,5,6,7 : 自定义时间，每个数代表星期的某一天

.start

object

定时开始时间及操作

.end

object

定时结束时间及操作

countdown_timer

object

倒计时定时器

.remaining

integer

倒计时读秒, N秒后执行倒计时

注解

0 和 1 指第一个和第二个开关，兼容了多控开关，若是单控开关则只需使用 0 即可

小技巧

power_state 、 timing_timer 、 countdown_timer 均为 可选 , 一旦使用, 则其子项都为 必选
