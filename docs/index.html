正泰智能云开发指南¶

  • API 被设计为符合 HTTP，REST 规范。例如：查询请求使用 Get方法，提交请求使用 Post 方法。如一个请求不是相应的 HTTP方法，将返回错误。根据业务需求，目前只采用Post方法。

  • 如无特殊说明，调用参数值应转码为：UTF-8, URL编码

  • URL编码 - WikiPedia定义
